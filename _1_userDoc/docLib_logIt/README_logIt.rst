Description
===========

    Permet de creer un fichier journal. Ce journal ce presente sous la forme suivante : ::
    
        'Titre de l'action'
         =================
    
        Descriptif
        
        [ TimeCode ] : action effectuee
        
        ################################################################################
        
    **N.B** : La library 'logging' fait déjà tous ça bien mieu.

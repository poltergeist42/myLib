f_dicoFuncRun(v_dicoValueRead)¶

f_dicoFuncRun( dict.value )

Cette methode execute une fonction en interpretant la valeur passer en argument. Cet argument placer en index dans le dictionnaire ‘d_dicoFunc’. c’est donc la clef. Pour chaque clef est assocé une valeur qui correspond a une fonction. Seule le nom de la fonction doit etre precisee, les parentheses ne doivent etres specifees.

N.B : La fonction doit etre declaree avant. N.B 2 : Il faut aussi remplir le dictionnaire avant d’appeler cette fonction.

ex :

def maFonction () :
    print( "Vous etes dans 'maFonction'" )

d_dicoFunc = { "maClef" : maFonction }

dicoFuncRun( "maClef" )

>>> Vous etes dans 'maFonction'

f_openRWFile(v_mode='r')¶

f_openRWFile( str, str )

Permet d’ouvrir un fichier en mode ‘Lecture’ : ‘r’ (valeur par défaut) ou en mode ‘ecriture’ : ‘w’

Les nom et chemin pour acceder / creer le fichier est donne par v_nomPlusChemin qui prend les valeurs par defaut de ‘f_setFileName()’ si aucun parametre n’est donne.

N.B : L’instance du fichier devra etre fermee avec ‘.close()’ depuis l’endroit de l’appel de ‘f_openRWFile()’
